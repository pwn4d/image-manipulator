#  image manipulator







## Installation
```bash
git clone https://github.com/pwn4d/image-manipulator.git
```
```
cd image-manipulator
```


## Usage
"h" or "horizontal" / "v" or "vertical" to change the flip type
```
~ im input.png output.png v
Image Flip Success
Image Inversion Success
```
